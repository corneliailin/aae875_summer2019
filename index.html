AAE875 Fundamentals of OOP and Data Analytics

Lectures: TH 113 on Tu, Th 2:00 - 5:00pm

Labs: CS 1370 on Wed 2:00 - 5:00pm

Description

In the first part of the class students will learn to define and utilize fundamental programming topics including variables and expressions, data types, loops, functions, how to organize a program using classes, basic testing and debugging techniques, IO processing etc. Additional topics include an introduction to good programming practices including consistent style and meaningful code comments, version control (Git, GitHub). The second hald of the class will introduce students to topics in data analytics, such as data visualization (incl. Jupiter and JupiterLab notebooks), descriptive statistics, linear regression, and data mining.

This class meets for two 3 hours class periods each week plus a weekly 3 hours lab over the semester.

There will be one Final Programming assignment (FP) related to applied economics that will explore all these topics.

  • GitHub provides hosting for files development version control using Git. Sign up for an account here.

  • Jupier and JupierLab notebooks are great tools for interactive data visualization. You can install them in your computer using pip or Anaconda. More information here.

Grading

Final grades will be determined by computing the weighted average of Participation, zyLabs, Final Program (FP), Exam 1 and Exam 2. Final letter grades may be curved upward, but a minimum guarantee is made of an A for 93 or above, AB for 87 or above, B for 80 or above, BC for 75 or above, C for 70 or above, and D for 60 or above.
